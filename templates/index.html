    Getting started

    API key generation through email involves sending an email request to the API provider to generate a new API key. The provider will verify the request and generate a new key, which will be sent to the user's email address.

    Validate Phone

    This API allows you to check the validity of a phone number. The API accepts a phone number as input and returns a JSON object indicating whether the phone number is valid or not. The API uses a POST request method and requires an api-key header for authentication. The request body should contain the phone number to be checked in the number field in JSON format. The response will be a JSON object containing a valid field indicating whether the phone number is valid or not. /check/phone_number 

    
          curl -X 'POST' \
          'https://api.tensangna.online/check/phone_number' \
            -H 'accept: application/json' \
            -H 'api-key: your key here' \
            -H 'Content-Type: application/json' \
            -d '{
                 "number": "+233208997096"
                }'

    If the phone number is valid, the API will return a JSON object with the following properties: valid (boolean) 

    
          "valid": true,
          "country": "Ghana",
          "countryCode": "+233",
          "phoneNumber": "208997096",
          "carrier": "Vodafone",
          "timeZone": [
            "Africa/Accra"
          ]

    Ip Address Checker

    Our IP checker API! It provides fast and accurate geolocation data and other information about any IP address. Our API is easy to integrate and can help enhance your applications with location-based data. /ip/verify 

    
          curl -X 'POST' \
          'https://api.tensangna.online/ip/verify' \
          -H 'accept: application/json' \
          -H 'api-key: your key here' \
          -H 'Content-Type: application/json' \
          -d '{
              "ip": "8.8.8.8"
            }'
